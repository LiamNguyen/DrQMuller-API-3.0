# DrQMuller-API-3.0

#### Global dependencies:
 - yarn ([yarnpkg.com](https://yarnpkg.com))

## Database

 - Replace current connection string in: `{ROOT}/config/environmentConfig.js` default environment

## Run

After cloning GIT repo, run: 
 - `yarn`
 - `npm i -g nodemon`
 - `nodemon`

## RAML

- `nodemon`
- Access via browser: `http://localhost: 5000/docs`
